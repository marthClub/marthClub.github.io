Setting up your Chromebook or other device for local development (independent from GitHub).
  • Pros
    • Changes you make will be reflected instantly instead of having to wait for GitHub commits to take effect.
    • The Caret editor is superior to the online editor in GitHub in many ways.
    • You can use "undo" in the Caret app to backtrack if you accidentally screw something up in your code.
  • Cons
    • Your project will only run on your own device. It won't be live on the internet for everyone to see. (However, you can very easily upload your sketch.js file to a web-enabled GitHub repository later)
  • Links:
    • Caret (Chrome Web Store) : A text editor designed for coding that can run on your Chromebook. It can open and save files in your local storage or your Google Drive.
    • Web Server for Chrome (Chrome Web Store) : An app that allows you to open web pages stored on your device or in your Google Drive as if they were published on the web.
    • "Blank Sketch" Files: The three files needed to run a p5.js sketch. Download them and then move them into a directory on your device or in your Google Drive.
      • index.html
      • p5.min.js
      • sketch.js
  • Steps:
    1. Download and launch the two apps (Caret and WSfC).
    2. Download the files in the "Blank Sketch" shared folder. It is recommended that you create a folder on your device or your Google Drive to store these in.
    3. In WSfC, click "CHOOSE FOLDER", and navigate to the folder where you stored the "Blank Sketch" files. Click "OK".
    4. In WSfC, click the blue link under "Web Server URL(s)". This will open the directory in your browser. Click the "index.html" to open your sketch. Note that it will just be a blank page until you put some code in the sketch.js file.
    5. In Caret, open the sketch.js file, and put in some code. Save using the menu, or press Ctrl-S.
    6. In your browser, refresh the webpage to see the changes. Repeat these last two steps until you've made something awesome!

The sites listed here will help you get started learning to code.
  • Khan Academy's Intro to JS : This is an excellent place for beginners to start. The interactive lessons will take you through some very important programming concepts. Everything is visually based, so you can see what your code is doing. There is excellent feedback, so you never get stuck.
  • The p5.js website : P5.js is a JavaScript library designed "to make coding accessible for artists, designers, educators, and beginners." It is the backbone of the creations we make in MARTH Club. There is a ton of great info on this site, including blocks of working code that you can copy and paste into your projects. Especially check out the tutorials and reference pages.

COLOR
Colors can be defined using the rgb format, which stands for red-green-blue. You can look up the rgb values using a color picker, or experiment yourself. Values range from 0 to 255.
  • background( r , g , b );
    • Sets the background color.
    • r, g, and b can range from 0 to 255.
    • Example: background( 255 , 0 , 0 );
      • Sets the canvas background to red.
    • Experiment with placing. Try placing the instruction:
      • at the end of the setup() function.
      • at the beginning of the draw() function
      • At the end of the draw() function. Note: this will draw the background over any shapes you’ve drawn, so you won’t actually see anything, unless you have the alpha value set low (section on transparency).
  • fill( r , g , b );
    • Sets the fill color for any shapes drawn.
    • r, g, and b can range from 0 to 255.
    • Example: fill( 0 , 255 , 0 );
      • Sets the fill color to green.
    • Place in the setup() function or anywhere in the draw() function before the shape-drawing instruction.
  • noFill()
    • Shapes drawn will not be filled.
    • No arguments.
    • Place in the setup() function or anywhere in the draw() function before the shape-drawing instruction.
  • stroke( r , g , b );
    • Sets the stroke color for any shapes drawn. This lines and the borders of shapes.
    • r, g, and b can range from 0 to 255.
    • Example: stroke( 0 , 0 , 255 );
      • Sets the fill color to blue.
    • Place in the setup() function or anywhere in the draw() function before the shape-drawing instruction.
  • noStroke()
    • Strokes will not be drawn. Lines are not drawn at all, and shapes have no borders.
    • No arguments.
    • Place in the setup() function or anywhere in the draw() function before the shape-drawing instruction.
    • Note that if noStroke() and noFill() are used together, no shapes or lines will be visible.
TRANSPARENCY
All instructions that take rgb arguments can take a fourth argument added, called alpha
  • Alpha defines how transparent something is.
  • Ranges from 0 to 255 just like the color values.
    • 0 = completely transparent.
    • 255 = completely opaque.
    • values in between are partially transparent.
  • Example: fill( 0 , 0 , 0 , 50 );
    • Sets the fill color to black with an alpha value of 50 (mostly transparent).
  • Some things to try:
    • Play with the alpha value on fill() and stroke() instructions.
    • Try using a small value (like 20) for the alpha in your background() instruction. Then move the instruction to the beginning of the draw() function.
SHAPES
For a list of shape instructions, visit the p5.js reference page, and look for a section called "2D Primitives". If you want to draw more complicated shapes, check out the instruction called beginShape().

GETTING STARTED PART 1: Create a GitHub account

Ms. Tortoise walks Achilles through the process of creating a GitHub account and explains a little about what GitHub is.

Achilles: OK, I'm ready.

Tortoise: Let's begin, then. In this first Part, we're going to get you set up with a GitHub account.

Achilles: What's GitHub?

Tortoise: GitHub is a service that allows people to create coding projects. It's used by all sorts of people, from students to hobbyists to professional programmers. It has a lot of useful features, but we're only going to focus on a couple to start out.

Achilles: Sounds good. How do I use it?

Tortoise: Well first you're going to need to create an account. I'll walk you through it and explain some things along the way. When there's something you need to do, I'll talk like this . You know, just to get your attention in case you zone out or something.

Achilles: Yeah I do that sometimes. What should I do first?

Tortoise: Go to github.com and click the green "Sign up" button.



Achilles: Got it. I'm at the GitHub page, and I clicked "Sign up".

Tortoise: Great. Now you're going to need to come up with a username and password.

Achilles: Should I use Achilles as my username?

Tortoise: It's never a good idea to use your actual name for a username on the internet.

Achilles: OK, I'll make something up. Oh! I have a really funny idea for a username!

Tortoise: Now, Achilles, make sure that you pick something appropriate.

Achilles: Why? Nobody is going to know it's me, right?

Tortoise: Here's the deal. One of the cool things about GitHub is that it can serve as your personal portfolio of your coding creations. A lot of people list their GitHub account on job and college applications because it gives a great snapshot of a person's abilities and growth in coding. You never know where this coding adventure will take you, so don't pick a username that might make you blush when it is brought up in an interview years from now.

Achilles: Ok, ok. I doubt I'll ever use this in a professional setting, but just in case I'll pick something "responsible."

Tortoise: Great. When you're ready, enter your new username, your current e-mail address, and your new password, and click "Create an account"



Achilles: OK. Now they're asking me to choose my plan.
